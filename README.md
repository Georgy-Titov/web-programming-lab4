# Web-программирование. Лабораторная работа №4 — Прогноз погоды

Учебный проект, разработанный в рамках дисциплины **«Web-программирование»** в Университете ИТМО.

## Описание проекта

Проект представляет собой веб-приложение «Прогноз погоды», в котором пользователь может получать актуальный прогноз погоды для своего текущего местоположения или выбранных городов. Приложение выполнено на чистом JavaScript, HTML и CSS без использования сторонних фреймворков.

## Основной функционал

* **Геолокация** — при первом открытии приложения запрашивается текущее местоположение пользователя для отображения прогноза на сегодня и минимум на два следующих дня.

* **Добавление городов** — возможность вручную добавить города для получения прогноза, с выпадающим списком автодополнения и валидацией.

* **Прогноз на 3 дня** — отображение погоды на сегодня и следующие два дня, включая температуру, погодные условия, ветер и осадки.

* **Обновление данных** — кнопка «Обновить» позволяет заново запросить информацию о погоде без перезагрузки страницы.

* **Удаление городов** — возможность удалить любой добавленный город.

* **Сохранение состояния** — все добавленные города и выбранное местоположение сохраняются в localStorage, что позволяет восстановить состояние приложения при перезагрузке.

* **Адаптивный интерфейс** — корректное отображение приложения на разных устройствах и экранах, без наложения элементов интерфейса.

* **Обработка ошибок** — информирование пользователя о недоступности геолокации или некорректно введенных городах.

## Используемые технологии

* **HTML** — структура страницы и контейнеры для отображения данных о погоде.  
* **CSS** — оформление интерфейса, карточек, анимации кнопки и адаптивный дизайн.  
* **JavaScript** — логика приложения, работа с API, обработка геолокации и ввода пользователя.  
* **Fetch API** — получение данных о погоде и геокодировании городов.  
* **LocalStorage API** — сохранение выбранных городов и текущей локации.

## Архитектура проекта

* `index.html` — базовая HTML-страница с подключением стилей и скрипта.  
* `style.css` — стили интерфейса, оформление карточек и адаптивность.  
* `script.js` — логика приложения, запросы к API, обработка событий и работа с LocalStorage.
